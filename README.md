# Microblog Stream

Microblog Stream is a WordPress theme that turns your posts into a fast, scrollable social style stream with simple replies. It is built for people who want a clean microblog experience without titles or heavy blog layouts.

> Status: Early development. Theme files and releases coming soon.

---

## Features

- Microblog style front page that reads like a social feed  
- Minimal post UI built around content, author, and timestamp  
- Single post view that matches the stream so nothing feels disconnected  
- Clean dark layout with accent color highlights  
- Reply focused commenting so readers can respond to individual posts

More details and screenshots will be added as development continues.

---

## Requirements

- WordPress 6.0 or higher  
- PHP 7.4 or higher

---

## Installation

Theme files are not published in this repository yet. Once the first release is ready you will be able to:

1. Download the latest release ZIP from the Releases page.  
2. Upload it in **Appearance → Themes → Add New → Upload Theme**.  
3. Activate **Microblog Stream**.  

Manual installation instructions for cloning from Git will also be added later.
